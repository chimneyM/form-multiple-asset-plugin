# om-app-plugin-react-boilerplate

使用 react 开发 onemind 应用插件的 boilerplate 工程

## 支持功能

* 开发预览
* 一键打包 （npm run plugin）
* css / less / scss 支持
* antd 内置，并在打包时剥离 antd.css 以减小包体积
* 接口封装，支持开发请求代理
